LOOPING STATEMENT

1)For.....loop

this loop executes statements as long as condition becomes true,when condition becomes false the statement come out from loop.

Benifit: it combines initialization(variable declaration), condition and iteration(increment or decrement)in single statement.

SYNTAX: for(initialization;condition;iteration)

{ statement block; }

this loop executes statements as long as condition becomes true,when condition becomes false the statement come out from loop.

SYNTAX: Initialization;
